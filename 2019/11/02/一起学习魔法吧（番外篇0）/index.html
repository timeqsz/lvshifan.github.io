高阶函数

高阶函数是什么？在第一次遇见这个名词的时候，我还以为是封装好的轮子，只是可以实现的功能比较强大，然而其实际的含义可以解释为\

  • 接受一个或多个函数作为输入

  • 输出一个函数

    满足上述两个条件之一便可称之为高阶函数
